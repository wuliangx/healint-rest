Store Orders REST API

This page lists all the REST API available for you to perform

  1. Get all orders

                    GET localhost:9000/orders

  2. Get specific order (for example, order with id 8)

                    GET localhost:9000/orders?id=8

  3. Create order

                    POST localhost:9000/orders Content-Type: application/json
    Example Body: 
    {
    "orderId": "CA-2016-152156",
    "orderDate": "08.11.16",
    "shipDate": "11.11.16",
    "shipMode": "Second Class",
    "quantity": 2,
    "discount": "0",
    "profit": "41.9136",
    "productId": "FUR-BO-10001798",
    "customerName": "Claire Gute",
    "category": "Furniture",
    "customerId": "CG-12520",
    "productName": "Bush Somerset Collection"
}

  4. Update Order (for example, order with id 8), only specify the fields you wish to modify in the json.

                    PUT localhost:9000/orders/8 Content-Type: application/json
    Example Body: 
    {
    "shipMode": "Third Class"
}
